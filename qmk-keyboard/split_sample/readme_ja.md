# split_sample 日本語

解説用のプロジェクトですが、Helixをベースに作成しています。

現在のところ、実際にテストはできていないのでご了承ください。

qmk_firmware のビルド環境の準備ができたら、以下を実行することでビルドすることができます。

    cd (qmk_firmwareのディレクトリ)
    cp -r (このファイルがあるディレクトリ) keyboards/
    make split_sample:default
